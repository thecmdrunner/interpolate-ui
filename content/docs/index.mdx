# What is <span className="text-[#9ECBFF] font-bold">Interpolate UI?</span>

It is a collection of **animated** components that you can use in any <a href="https://remotion.dev" referrerPolicy="no-referrer" target="_blank" rel="noopener noreferrer" className="font-bold leading-[0.4]"> <img src="https://github.com/remotion-dev/brand/raw/main/withouttitle/element-0.png" className="!h-4 inline-block w-4 aspect-square" /> Remotion</a> project.

- Built with **Typescript** and **TailwindCSS**.
- **Beautiful** by default, **customizable** when needed - the code lives in your repository, no install required.
- We aim to help you convert your ideas into a Remotion video as quickly as possible.

## Get started

1. Install the most common packages beforehand if you prefer:
